# markdown书写

## 定义文字颜色

源码：

```
<font color="red">这是红色字体 </font>
```

效果：

<font color="red">这是红色字体 </font>

## 锚点功能

源码：

```
# titleA
# titleB

[跳转到titleA](#titleA)
[跳转到titleB](#titleB)
```

效果：

### titleA

#### titleB

[跳转到titleA](#titleA)
[跳转到titleB](#titleB)

> typora中需要 ctrl + click ，跳转到指定位置
>
> 在`web`环境下，不需要`Ctrl+Click`，自动就能滑动页面到指定位置。如果是`Ctrl + click`的话，就是打开新页面，并跳转到页内指定位置。
